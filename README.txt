Provide professional support to your users with IssuePress. 

== Description ==

IssuePress turns your WordPress site into a support platform. Enables users to create support requests and gives you the power to answer those questions and provide professional customer service.

Features:

- Adds the Support Request Custom Post Type
- Support Request Sections, Labels, and Status
- Extensible templating system that works great with themes and child-themes
- Shortcodes for easy usage
- Base IssuePress Widgets
- Highly hookable structure
- Compatibility with IssuePress Extensions that add custom functionality


== Installation ==

*From your WordPress dashboard*

1. Visit 'Plugins > Add New'
2. Search for 'IssuePress'
3. Activate IssuePress from your plugins page

*From WordPress.org*

1. Download IssuePress
2. Upload the IssuePress directory to your '/wp-content/plugins/' directory
3. Activate the plugin through the 'Plugins' menu in WordPress

For more info on installtion, read about it on the [WordPress Codex](http://codex.wordpress.org/Managing_Plugins#Installing_Plugins).


== Frequently Asked Questions ==

= Where is the official documentation? = 

You can read the official IssuePress documentation [here](http://issuepress.co/docs/).

= I'm having an issue, where can I get help? =

Great question, we're happy to help you get started with IssuePress. Just visit our support section at http://issuepress.co/support/

= What's IssuePress Core =

IssuePress Core is the foundation for a customer support solution, it provides that base functionality that will help your provide professional support to your users.

= What's an IssuePress Extension =

 We're adding extensions that add will customize IssuePress to typical support needs that you might need, but that some people might not find as useful.
